üìò Dokumentasi IndoPost API

IndoPost API adalah REST API yang menyediakan data wilayah administratif Indonesia secara hierarkis, mulai dari provinsi hingga kelurahan beserta kode pos.

Dokumentasi ini ditujukan untuk pengguna umum yang ingin memahami cara kerja API sebelum melakukan registrasi.

üîß Cara Kerja API

  1. User melakukan registrasi
  2. User login dan mendapatkan API Key
  3. API Key digunakan untuk mengakses endpoint
  4. Data diambil secara bertingkat (provinsi ‚Üí desa)

üåê Base URL

http://localhost:5000/api

üîê Authentication

Semua endpoint API membutuhkan API Key yang dikirim melalui header: 

Authorization: Bearer YOUR_API_KEY

üåç Hirarki Data Wilayah

Provinces ‚Üí Regencies ‚Üí Districts ‚Üí Villages

Data wilayah harus diambil secara berurutan menggunakan kode dari response sebelumnya.

GET /api/provinces GET

Mengambil seluruh daftar provinsi di Indonesia.

http://localhost:5000/api/provinces
{
  "data": [
    {
      "code": "34",
      "name": "DAERAH ISTIMEWA YOGYAKARTA"

GET /api/villages GET

QueryWajibContoh

http://localhost:5000/api/villages?district_code=347104
