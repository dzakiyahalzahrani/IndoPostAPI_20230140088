REST API data wilayah Indonesia

Base URL 
http://localhost:5000/api

Semua endpoint wajib API Key

Authorization: Bearer YOUR_API_KEY

GET /api/provinces GET

Mengambil seluruh provinsi.

Headers 
Authorization: Bearer YOUR_API_KEY
Query 
-
Request URL 
http://localhost:5000/api/provinces
cURL 
curl -X GET http://localhost:5000/api/provinces \
-H "Authorization: Bearer YOUR_API_KEY"

Response

{
  "data": [
    {
      "id": 34,
      "code": "34",
      "name": "DAERAH ISTIMEWA YOGYAKARTA"
    }
  ]
}

GET /api/regencies GET

Kabupaten / kota berdasarkan provinsi.

Query 
province_code (wajib)
Contoh 
province_code=34
Request URL 
http://localhost:5000/api/regencies?province_code=34
cURL 
curl -X GET "http://localhost:5000/api/regencies?province_code=34" \
-H "Authorization: Bearer YOUR_API_KEY"

Response

{
  "data": [
    { "code": "3401", "name": "KULON PROGO", "type": "KABUPATEN" },
    { "code": "3471", "name": "YOGYAKARTA", "type": "KOTA" }
  ]
}

GET /api/districts GET

Kecamatan berdasarkan kabupaten/kota.

Query 
regency_code=3471
Request URL 
http://localhost:5000/api/districts?regency_code=3471

Response

{
  "data": [
    { "code": "347104", "name": "UMBULHARJO" }
  ]
}
